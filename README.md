# dotfiles

The strongest configuration I thought up

## Environment

macOS iTerm2 (Terminal)

## Usage

### macOS

#### memo

Install apps:

- Google Japanese Input
  - preferences > IME
  - preferences > Keyboard shortcuts
- Google Chrome
- 1Password
- iTerm2
- Alfred

#### Configure

```sh
cd ~
git clone 'https://github.com/tsukker/dotfiles.git'
cd dotfiles
sh ./macos.sh
```

#### Neovim

```vim
:PlugInstall
:PlugStatus
```
